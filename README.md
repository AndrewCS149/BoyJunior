# Boy-Junior
Boy-Junior is an ongoing project that allows me to learn the basics of C# and the MonoGame framework. It is an open-world, 2D, top-down game that allows the user to roam freely.

### Player Controls:
  * Move left: 'A'
  * Move right: 'D'
  * Move up: 'W'
  * Move down: 'S'

### Computer Prerequisits:
  * MonoGame
  * MonoGame.Extended

---

Language: C#

Framework: MonoGame
